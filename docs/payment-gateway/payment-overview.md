# Payment System Overview

This document describes the design of the payment system used to process order payments through Paymob.

The goal of this design is to provide a secure and reliable payment workflow that integrates with external payment providers while maintaining proper validation and security checks inside the system.

The payment system includes:

- Payment initiation
- Communication with Paymob
- Payment verification
- Webhook handling
- Payment status tracking

The system ensures that payments are validated and properly linked to user orders.
